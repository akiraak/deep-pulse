# 新NISAの上限額と限度額 {#新nisaの上限額と限度額 .s1-hdg-lv1__element}

<figure class="mb-normal">
<img src="images/main.png" alt="新NISAの上限額と限度額" />
</figure>

2024年1月から、新NISA制度がスタートしました。新NISAでは、旧NISAよりも投資限度額が拡大し、非課税保有期間が無期限化されたため、より長期的に非課税投資ができるようになりました。\
新NISA制度の投資上限額と非課税保有限度額についてご説明します。

## 新NISAの上限額・限度額は？ {#新nisaの上限額限度額は .s1-hdg-lv2__element}

NISAは2024年1月に制度変更があり、年間投資枠の上限や非課税保有限度額が変更になりました。\
2024年1月スタートの新しいNISA制度（以降新NISAと表記）では、年間投資枠の上限が360万円、非課税保有限度額は生涯で1,800万円となり、2023年までの旧NISA制度と比較して、投資限度額・非課税保有限度額が大幅に拡大しました。\
非課税で運用商品を保有できる期間が無期限化されたことも大きな変更点です。

+------------------+--------------------+-------------+-------------------------------+-------------+
|                  | 旧NISA（～2023年） |             | 新NISA（2024年～）            |             |
+==================+====================+=============+===============================+=============+
|                  | つみたてNISA       | 一般NISA    | つみたて投資枠                | 成長投資枠  |
+------------------+--------------------+-------------+-------------------------------+-------------+
| 制度選択         | 併用不可           |             | 併用可                        |             |
+------------------+--------------------+-------------+-------------------------------+-------------+
| 年間投資上限額   | 40万円             | 120万円     | 120万円                       | 240万円     |
+------------------+--------------------+-------------+-------------------------------+-------------+
| 非課税保有限度額 | 800万円            | 600万円     | 1,800万円                     |             |
|                  |                    |             |                               |             |
|                  |                    |             | -   成長投資枠は1,200万円まで |             |
+------------------+--------------------+-------------+-------------------------------+-------------+
| 非課税保有期間   | 最長20年           | 最長5年     | 無期限                        |             |
+------------------+--------------------+-------------+-------------------------------+-------------+

### 新NISAの年間投資上限額 {#新nisaの年間投資上限額 .s1-hdg-lv3__element}

新NISAでは、つみたて投資枠と成長投資枠の2つの枠があり、それぞれ併用が可能です。\
年間投資枠とは、1月～12月の期間での投資可能金額を指し、上限360万円の内訳は、つみたて投資枠で120万円、成長投資枠で240万円と決められています。\
どちらかの投資枠のみで上限360万円の投資はできません。また、年間投資枠は利用の有無に関係なく毎年リセットされるため、使わなかった年間投資枠を翌年以降に繰り越すことはできません。年間投資枠をフル活用したい方は注意が必要です。

NISA年間投資枠を使い切るには

-   つみたて投資枠\
    「NISAつみたて投資枠使い切り設定」をご利用いただくことで、年間120万円の投資枠を使い切ることが可能です。\
    [「NISAつみたて投資枠使い切り設定」の詳細を見る](/web/nisa/reserve/tsukaikiri_setting/){.s1-nisa-link}
-   成長投資枠\
    積立金額の調整やスポット購入により、年間240万円の投資枠を使い切ることが可能です。

### 新NISAの非課税保有限度額 {#新nisaの非課税保有限度額 .s1-hdg-lv3__element}

新NISAでは、年間投資枠に加えて、生涯における非課税保有限度額も決められています。\
非課税保有限度額とは、生涯においてNISA口座全体で非課税保有できる金額の上限を指し、購入時の金額（簿価）で管理されます。\
非課税保有限度額は総枠で1,800万円ありますが、これは成長投資枠とつみたて投資枠の合算での限度額です。\
つみたて投資枠を使わず、成長投資枠のみを利用する場合は非課税保有限度枠が異なり、1,200万円が上限となりますので注意しましょう。\
すべての投資枠を活用するにはつみたて投資枠と成長投資枠を併用する必要があります。

### 非課税保有限度額（総枠）の再利用 {#非課税保有限度額総枠の再利用 .s1-hdg-lv3__element}

新NISAでは保有商品を売却した場合、翌年以降に枠の再利用ができるようになりました。\
非課税保有限度額まで投資をしてしまった場合も、売却により翌年のNISA枠として投資が可能です。\
再利用できる枠は売却時点の金額ではなく、取得時点の金額となります。仮に100万円で購入した商品を80万円や120万円で売却した場合も、翌年再利用が可能となる枠は購入時の金額の100万円です。利益分や損失分は考慮されません。\
再利用する場合でも、年間投資上限額の360万円を超えて投資することはできませんので、ご注意ください。

-   [詳細を確認する：NISA（成長投資枠・つみたて投資枠）では非課税枠を再利用できるというのはどういうことですか？](https://faq.rakuten-sec.co.jp/90000691){target="_blank"}

## 新NISAの非課税保有期間は？ {#新nisaの非課税保有期間は .s1-hdg-lv2__element}

2024年からの新NISAでは非課税保有期間が無期限となりました。\
2023年までの旧NISAの非課税保有期間はつみたてNISAが20年間、一般NISAが5年間だったため、非課税保有期間を気にすることなく、より長期的に非課税投資ができるようになりました。

## 2023年までの旧NISA・つみたてNISA投資分はどうなる？ {#年までの旧nisaつみたてnisa投資分はどうなる .s1-hdg-lv2__element}

つみたてNISA・一般NISA・ジュニアNISAは2023年で制度終了となりましたが、2023年までの旧NISA口座で保有している商品には、引き続き旧NISA制度のルールが適用されます。\
そのため、つみたてNISAは購入時から20年間・一般NISAは5年間・ジュニアNISAは18歳になるまで非課税で保有が可能です。ロールオーバー（翌年の非課税投資枠に移管すること）の利用も2023年までのため、非課税保有期間が終了した場合は売却するか課税口座へ移すことになります。\
また、旧NISA口座で保有している商品は、新NISAの非課税保有限度額（総枠）の1,800万円とは別のものとして分けて管理されます。

旧NISA制度からの変更点をさらに確認したい方は、NISA制度について説明しているページをご覧ください。
