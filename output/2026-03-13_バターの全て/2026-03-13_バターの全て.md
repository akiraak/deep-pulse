# バターの全て — 歴史・科学・経済・健康を徹底解剖

> 紀元前2000年から人類と歩んできた「黄金の固形油脂」のすべてを紐解く

---

朝のトーストにひと塗り、フランス料理のソースにひとかけ、焼き菓子の生地にたっぷり。バターは私たちの食卓に欠かせない存在だ。しかし、「バターは体に悪い」「日本のバターは高すぎる」「マーガリンのほうがヘルシー」——こうした話を耳にするたびに、本当のところはどうなのだろうと気になったことはないだろうか。

この記事では、バターの歴史から製造方法、種類、栄養と健康、マーガリンとの比較、料理の科学、世界市場の動向、そして日本特有の「バター不足」問題まで、バターにまつわるあらゆるテーマを一挙に解剖する。例えるなら、バターという一つの食材を「顕微鏡」で覗き込むような、深くて広い旅だ。

## バターの歴史 — 紀元前2000年から現代まで

### 古代: 薬品・化粧品だったバター

バターの起源は定かではないが、[インドの古い経典には紀元前2000年ごろ](https://www.j-milk.jp/findnew/chapter3/0102.html)、すでにバターらしきものがつくられていたという記述が見られる。旧約聖書の中にも「かくてアブラハムはバターを取り、乳を取り......」という一節があり、古くからバターが存在していたことがわかる。

紀元前500年ごろ、ギリシャの歴史家[ヘロドトスは「馬や牛の乳を木の桶に入れ、激しく振動させ、表面に浮かび上がった部分をすくい取ってバターをつくった」と書き残している](https://www.j-milk.jp/findnew/chapter3/0102.html)。古代アラビアでも革袋に乳を入れ、それを振動させてバターを製造していた。

例えるなら、古代のバター作りは「ペットボトルに生クリームを入れて振り続ける」ようなもので、原理自体は現代とほとんど変わっていない。

興味深いのは、古代ギリシャやローマ時代にはバターは食糧としてよりも**医薬品や化粧品**として用いられていたことだ。当時のローマ帝国では、[ゲルマン人が食べていたバターを「野蛮で貧しい者が食べるもの」と考えていた](https://www.olive-hitomawashi.com/column/2020/03/500.html)という。フランス語には「Compter pour du beurre（バターに相当する＝重要性を持たない）」という表現があり、かつてバターが価値の低いものだった名残が言葉に残っている。

### 中世ヨーロッパ: 権力の象徴へ

[7世紀ごろのヨーロッパでは、バター、牛脂、豚脂など食用油の価値が高まり](https://www.olive-hitomawashi.com/column/2020/03/500.html)、上流階級の富や権力を表す象徴となった。食用油をたっぷりと使った料理で宴を開き、おもてなしをしたのだ。農民から年貢や地代としてバターを納めさせることもあり、乳から作ることができるバターはわずかだったため、農民は苦しい生活を強いられていた。

バターは宗教とも深く結びついていた。中世イングランドでは結婚した夫婦に「たくさんの子どもが授かりますように」と願いを込めてバターの入った壺を贈り、フランスのブルターニュでは子孫繁栄を願ってバターに彫刻を施し結婚式の装飾にした。インドでは溶かしバター「ギー」がヒンドゥー教の儀式に用いられ、チベットではバターを仏像に塗る風習がある。

### 日本への伝来: 「酥」から「雪印」まで

日本では、[孝徳天皇（在位645〜654年）の時代に、百済から渡来した智聡一族によって牛乳が天皇家に献上されている](https://www.j-milk.jp/findnew/chapter3/0102.html)。このとき、あわせて「酥（そ）」も献上された可能性があり、これはバターやバターオイルのようなものであったと考えられている。

醍醐天皇（在位897〜930年）の時代には、諸国に命じて蘇をつくり天皇に貢進させる「貢蘇の儀」も行われた。しかし、その後権力が武家に移ると乳利用の文化は廃れてしまった。乳製品がふたたび日本にやってくるのは18世紀以降、長崎の出島のオランダ商館でのことだ。

日本でバターの製造が始まったのは明治時代からである。[1872年、東京麻布の北海道開拓使第3官園実習農場で試験的につくられ](https://nyukyou.jp/dairyqa/2107_131_435/)、本格的な製造は[1885年、東京麹町の北辰社がクリーム分離機と回転チャーンを導入してから](https://nyukyou.jp/dairyqa/2107_131_435/)だ。北海道では大正14年に[629人の酪農家が出資し北海道製酪販売組合が組織され、商標は「雪印」](https://www.meg-snow.com/fun/academy/butter/history/)。最初は木製の樽にハンドルをつけた手回しチャーンで手作りされていた。

| 時代 | 出来事 |
|------|--------|
| 紀元前2000年頃 | インドの経典にバターの記述 |
| 紀元前500年頃 | ギリシャのヘロドトスが製法を記録 |
| 7世紀 | ヨーロッパで食用油として価値が高まる |
| 645〜654年 | 日本に「酥」が伝来 |
| 1872年 | 日本初のバター試作（東京麻布） |
| 1885年 | 日本初の本格製造（北辰社） |
| 1925年（大正14年） | 雪印の前身組織が北海道で設立 |

## バターの製造方法と種類 — 「振る」から「科学」へ

### バターができるまで

バターとは、生乳などからつくられたクリームを攪拌し、生じた脂肪粒を集めて固め、練り上げたものだ。[乳等省令では「生乳、牛乳又は特別牛乳から得られた脂肪粒を練圧したもの」で「大腸菌群陰性」のものと定義され、乳脂肪分80.0％以上、水分17.0％以下](https://www.meiji.co.jp/meiji-shokuiku/know/lovable-milk/butter/)と定められている。

クリームには[脂肪球皮膜たんぱく質（MFGM）](https://www.meiji.co.jp/meiji-shokuiku/know/lovable-milk/butter/)に包まれた脂肪球が浮遊し、攪拌（チャーニング）すると脂肪球同士がぶつかりあい、MFGM膜が破れた脂肪滴が集まって固まる。例えるなら、シャボン玉（脂肪球）を何度もぶつけると膜が破れて中身が合体するようなイメージだ。

製造方法は紀元前の革袋から、木桶、陶器鉢＋ヘラ棒を経て、17世紀末に動力が導入された。現代の工場では大型の連続式チャーンが使われるが、「クリームを強く攪拌して乳脂肪の塊を集める」という基本原理は数千年前から変わっていない。

### バターの分類

バターは「製法」と「食塩」と「飼育方法」の3軸で分類できる。

| 分類軸 | 種類 | 特徴 |
|--------|------|------|
| **製法** | 非発酵バター（甘性バター） | 乳酸菌を加えずクリームをそのまま攪拌。クセがなくすっきり。日本で主流 |
| | 発酵バター | クリームに乳酸菌を加えて発酵させてから製造。香りとコクが強い。ヨーロッパで主流 |
| **食塩** | 有塩バター | 食塩1〜2%添加。保存性が高く、塩味がパンの甘さを引き立てる |
| | 食塩不使用バター | 食塩無添加。製菓用・調理用に最適。味の微調整がしやすい |
| **飼育方法** | 一般的なバター | 穀物混合飼料で育てた牛のミルクから製造 |
| | [グラスフェッドバター](https://www.kurashiru.com/articles/9c52752c-ba01-467a-af26-c7a1ffa44a96) | 牧草のみで育てた牛のミルクから製造。黄色が濃く、不飽和脂肪酸が豊富 |

### その他のバリエーション

- **ホイップバター**: バターに空気を含ませてふわっと軽くしたもの。カフェのパンケーキに乗っているのがこれだ。100gあたり552kcalと通常のバター（約700〜780kcal）より低カロリー
- **ギー（精製バター）**: バターから水分とたんぱく質を除いた純粋なバターオイル。焦げにくく高温調理に向く。インドのアーユルヴェーダでも重宝される
- **フレーバーバター**: ハーブ、ガーリック、レーズンなどを混ぜた風味付きバター
- **[カルピスバター](https://financierlove.com/types-of-butter/)**: カルピスの製造過程でできる脂肪分を使ったバター。生産数が少なく「幻のバター」とも。本場フランスのバターに最も近い味とされ、一流シェフが秘伝の味として口外しなかったという逸話がある

| バターの種類 | カロリー（100gあたり） | 価格帯 | 風味 |
|-------------|----------------------|--------|------|
| 発酵バター | 713kcal | 高い | 甘い風味＋さわやかな酸味と芳香 |
| 有塩バター | 700kcal | 手頃 | 乳の風味＋塩味 |
| 食塩不使用バター | 720kcal | 手頃 | 乳本来の風味 |
| グラスフェッドバター | 780kcal | 高い | あっさりとした味わい |
| ホイップバター | 552kcal | 手頃 | バターの風味で柔らかい |
| カルピスバター | 744kcal | 中程度 | コクがあるのにスッキリ |

## バターの栄養と健康 — 「体に悪い」は本当か？

### バターに含まれる栄養

[バター1箱（200g）には、約4.4リットルもの牛乳が使用されている](https://www.meg-snow.com/hokkaido-butter/knowledge/0003.html)。バターの約80％は脂質だが、食用油脂の中でもっとも消化がよく、[吸収率はなんと95%以上](https://www.meg-snow.com/hokkaido-butter/knowledge/0003.html)。胃腸の弱い方やお年寄りにもおすすめできる食品とされている。

| 栄養素 | 有塩バター（100gあたり） | 食塩不使用バター（100gあたり） |
|--------|------------------------|-----------------------------|
| エネルギー | 745kcal | 763kcal |
| たんぱく質 | 0.6g | 0.5g |
| 脂質 | 81.0g | 83.0g |
| ビタミンA | 520μg | 790μg |
| ビタミンD | 0.6μg | 0.7μg |
| ビタミンE | 1.5mg | 1.4mg |

バターの黄色みの正体はビタミンAのカロテンだ。カロテンは牧草に含まれる栄養で、肌や粘膜を健康に保ち、細菌に対する抵抗力を高める役割がある。また、カルシウムの吸収に役立つビタミンD、老化を防ぐビタミンEも含まれる。

例えるなら、バターは「少量で多くの栄養を届けるエネルギーカプセル」のようなものだ。

### コレステロールとカロリーの実態

[バター100gあたりのコレステロール量は約210mg](https://www.meg-snow.com/hokkaido-butter/knowledge/0003.html)。ただし、食パンにバターを塗る場合、使用量は5〜10g程度なのでコレステロール量は10.5〜21mgに過ぎない。厚生労働省の「日本人の食事摂取基準（2020年版）」によると、脂質異常症の重症化予防には200mg/日未満が望ましいとされており、普通に食べる限り心配はない。

カロリーについても、トースト2枚にバターを10g塗ったとしても74.5kcal程度。同量のオリーブ油やごま油と比べてむしろ低い数値だ。

### 最新研究: 22万人・33年間の追跡調査

2025年3月、米ブリガム・アンド・ウイメンズ病院のYu Zhang氏らの研究が[「JAMA Internal Medicine」に掲載された](https://www.carenet.com/news/general/hdn/60360)。医療従事者22万1,054人を33年間追跡した大規模研究で、以下の結果が示された。

- **バター摂取量が最も多いグループ**は最も少ないグループに比べて**全死亡リスクが15%高い**
- **植物油摂取量が最も多いグループ**は最も少ないグループに比べて**全死亡リスクが16%低い**
- 毎日**10gのバターを植物油に置き換える**と、**全死亡リスクおよびがんによる死亡リスクがそれぞれ17%低下**

Zhang氏は「毎日の食事でバターを植物油に置き換えることで全死亡リスクが17％低下するというのは、健康に対してかなり大きな影響」と述べている。

### 結局どうすればいい？

バターは目の敵にするものではなく、**メリハリが大切**だ。普段はオリーブオイルなどの植物油を中心に使い、バターは「ここぞ」という場面で楽しむ。例えるなら、バターは「毎日のジョギングシューズ」ではなく「特別な日のドレスシューズ」——使いどころを選ぶことで、健康も美味しさも両立できる。

## バター vs マーガリン — 終わらない論争

### 原料の違い

バターは牛乳（動物性脂肪）から、マーガリンは植物性油脂を主原料にして作られる。バターは生乳のクリームを攪拌して脂肪粒を集めたもの。マーガリンは植物油に乳化剤や着色料、香料などを添加して固形化したものだ。

例えるなら、バターは「搾りたてのフルーツジュース」、マーガリンは「果汁入り飲料」のような関係だ。原料も製法もまったく異なる。

### トランス脂肪酸の「逆転現象」

かつてマーガリンは「トランス脂肪酸が多い危険な食品」とされていた。しかし近年、[製造技術の改善によって状況が大きく変わった](https://www.miyoshi-yushi.co.jp/food/transfattyacid/)。

| 食品 | トランス脂肪酸（100gあたり） |
|------|----------------------------|
| **バター** | **1.9g** |
| **マーガリン** | **0.99g** |

現在のマーガリンに含まれるトランス脂肪酸は[**バターのおよそ半分**にまで低減されている](https://www.miyoshi-yushi.co.jp/food/transfattyacid/)。これは多くの人にとって意外な事実だろう。

ただし、バターに含まれるのは**天然由来**のトランス脂肪酸であり、[食品安全委員会のリスク評価書では「天然由来のトランス脂肪酸は、冠状動脈疾患との関係は低いと考えられる」としている](https://www.yotsuba-shop.com/Page/Feature/FeaturePage020.aspx)。一方、マーガリンに含まれるのは**工業由来**（油脂の加工・精製過程で生成）のトランス脂肪酸だ。

### 日本人はそこまで心配しなくていい

WHO（世界保健機関）はトランス脂肪酸の摂取量を総エネルギーの1%未満に抑えるよう勧告している。しかし[日本人の平均的な食生活では、トランス脂肪酸の摂取量はすでに総エネルギー比の**0.3〜0.6%程度**にとどまっており](https://www.lireclinic.com/column/%E3%83%90%E3%82%BF%E3%83%BC%E3%81%8B%E3%83%9E%E3%83%BC%E3%82%AC%E3%83%AA%E3%83%B3%E3%81%8B%EF%BC%88%E3%83%88%E3%83%A9%E3%83%B3%E3%82%B9%E5%9E%8B%E8%84%82%E8%82%AA%E9%85%B8%EF%BC%89/)、WHOの基準を大きく下回っている。

### 結局どちらを選ぶべきか？

| 観点 | バター | マーガリン |
|------|--------|-----------|
| 風味 | 天然のミルクの風味、コク | バターに似せた風味 |
| トランス脂肪酸 | 1.9g/100g（天然由来） | 0.99g/100g（工業由来） |
| 飽和脂肪酸 | 多い | 少ない |
| カロリー | 700kcal/100g | 715kcal/100g |
| 価格 | 高い | 安い |
| 保存性 | 短い | 長い |

結論としては、**どちらかが絶対に正しいということはない**。料理の風味を重視するならバター、日常的な使用でコストや保存性を重視するならマーガリン。健康面では、どちらも「適量」が鍵だ。

## バターの料理科学 — なぜバターを使うと美味しくなるのか

### メイラード反応 — 色・香り・味を同時に変える魔法

バターを加熱すると、含まれる乳糖（糖質）とタンパク質が[メイラード反応](https://savory.co.jp/column/maillard-reaction-cooking-tips/)を起こす。メイラード反応は120℃以上の温度帯から始まり、数百種類もの香り成分や褐色の物質（メラノイジン）が次々と生まれる。

この反応が料理をおいしくする理由は、「色」「香り」「味」という三つの要素を**同時に変化させる**ところにある。ステーキの焼き色、トーストの香ばしさ、クッキーのきつね色——これらはすべてメイラード反応の産物だ。

例えるなら、メイラード反応は「料理のフルメイク」。下地（色）、香水（香り）、リップ（味）を一度に仕上げてくれる。

### 焦がしバター（ブールノワゼット）

バターを加熱し続けると、乳糖＋タンパク質のメイラード反応が進み、褐色に変化して独特のナッツのような香りが生まれる。これが[「焦がしバター（ブールノワゼット）」](https://note.com/travelingfoodlab/n/n49274f243bae)だ。フランスでは中世から使われているソースで、マドレーヌなどの焼き菓子にも風味付けに使われる。

ただし、「焦がしバター」と「焦げたバター」は別物だ。メイラード反応が生み出す褐色物質と、炭化反応で生じる「焦げ」はまったく異なる化学反応である。

### バターが焼き菓子に欠かせない理由

バターは焼き菓子において以下の役割を果たす。

- **層状構造の形成**: パイやクロワッサンでは、冷たいバターの層が焼成時に溶けて水蒸気を発生させ、サクサクの層を作る
- **風味の付与**: 乳脂肪特有のコクと香りが菓子全体に広がる
- **食感の調整**: バターの脂肪分が小麦粉のグルテン形成を抑え、サクッとした食感を生む
- **メイラード反応の促進**: バターに含まれる乳糖とタンパク質が反応し、焼き色と香ばしさを生む

## 世界のバター消費と市場 — グローバル・バター・マップ

### 世界の1人当たり消費量ランキング

[FAO（国連食糧農業機関）の2021年統計](https://fumib.net/butter-consumption/)によると、世界のバター消費量にはきな格差がある。

| 順位 | 国 | 1人当たり年間消費量 |
|------|-----|-------------------|
| 1位 | バーレーン | 9.95kg |
| 2位 | ニュージーランド | 8.59kg |
| 3位 | フランス | 8.12kg |
| 4位 | デンマーク | 7.72kg |
| 5位 | ベルギー | 7.43kg |
| — | 世界平均 | 1.55kg |
| — | **日本** | **0.67kg** |

日本のバター消費量はフランスの**12分の1以下**、世界平均の半分にも満たない。例えるなら、フランス人が毎日パンに塗るバターの量を、日本人は1か月かけてやっと消費するようなイメージだ。

総消費量ベースでは、1位インド（5,392,000トン）、2位EU（2,185,000トン）、3位アメリカ（857,000トン）となっている。

### 世界市場規模と成長予測

[グローバルバター市場は2025年に469.5億ドル（約7兆円）規模と推定され、2031年には619.9億ドルに達する見込みだ（年率4.74%成長）](https://www.mordorintelligence.com/industry-reports/butter-market)。アジア太平洋地域が市場の約39%を占めており、インドや中国での需要拡大が成長を牽引している。

### 2024年の欧州バター危機

2024年、ヨーロッパでバター価格が急騰した。[EU加盟27か国平均で前年比19%の値上がり](https://www.euronews.com/business/2024/12/20/rising-butter-prices-leave-european-consumers-with-a-sour-taste)となり、特にフランスでは[卸売価格が2024年9月に**1トンあたり8,180ユーロ**と、前年の**4,260ユーロ**からほぼ倍増した](https://www.connexionfrance.com/news/french-bakeries-impacted-by-soaring-butter-prices/682530)。

原因は、家畜の疫病流行による牛乳生産の減少と、乳業メーカーがより利益率の高いチーズ生産を優先したことだ。フランスのパン屋は原材料費の急騰に直面し、クロワッサンなどバターを多用する製品の値上げを余儀なくされた。

### 2025年の急反転 — 価格崩壊

ところが2025年、状況は劇的に反転した。アメリカ、ヨーロッパ、オセアニアで牛乳の過剰生産が同時に発生し、需要の減退と重なって[価格が急落。2025年9月には年初から**22%下落**し、アメリカでは2021年11月以来の安値となるポンドあたり2ドル以下にまで落ち込んだ](https://dairynews.today/news/global-butter-prices-expected-to-decline-amidst-surplus-supply.html)。

2026年2月時点ではトンあたり4,600ユーロに回復しつつあるが、前年同月比ではまだ**約35%安い**水準だ。

例えるなら、バターの世界市場は「ジェットコースター」のような値動きを見せている。2024年の高騰から2025年の急落、そして2026年のゆるやかな回復——こうした乱高下の背景には、牛乳の生産量という予測困難な変数がある。

## 日本のバター事情 — なぜ日本のバターは高くて足りないのか

### 価格高騰の現状

[2020年の価格を100とすると、2025年12月時点でバターは**約140**（200gあたり613円）にまで上昇している](https://shokulab.unitecfoods.co.jp/article/detail285/)。関東生乳販連は2020年以降、生乳価格を計24円/kg引き上げており、末端消費者にとっては50〜60円/kg程度の値上げとなった。乳製品価格の低下に作用するような目ぼしい要素は現状ほとんどなく、価格は現行水準で推移すると予想されている。

### 構造的問題: ブレーキとアクセルを同時に踏む酪農政策

日本のバター不足は単なる一時的な問題ではなく、**半世紀にわたる構造的な問題**だ。

**生乳の用途優先順位**: 保存期間の短さから、生乳は**牛乳 → 生クリーム → チーズ → バター**の順で使われる。バター製造は常に「最後の候補」であり、牛乳需要が増えればバターに回す生乳が減る仕組みだ。例えるなら、バターは「電力供給で言えば最後に回される工場」のようなものだ。

**矛盾した政策**: 政府は一方で牛の削減に1頭15万円の補助金を出し（ブレーキ）、他方で「畜産クラスター事業」で酪農の拡大を支援している（アクセル）。東京大学の研究者は[「酪農は、ブレーキとアクセルを両方踏んでいる」と指摘している](https://agri.mynavi.jp/2023_12_05_247643/)。

**酪農家の減少**: [酪農家戸数は2005年の28,000戸から2015年には17,700戸へ約37%減少](https://www.nippon.com/ja/column/g00316/)。さらに、[酪農家が所得確保のため受精卵移植で和牛生産を増やした結果、乳牛となるホルスタインの雌が生まれにくくなっている](https://www.jacom.or.jp/niku/closeup/2025/250203-79299.php)。

### 輸入制度の壁: カレントアクセスという「鍵のかかった扉」

日本のバター輸入は、[独立行政法人農畜産業振興機構（ALIC）が一元管理している](https://www.maff.go.jp/j/chikusan/gyunyu/antei_kyokyu.html)。WTOの国際約束に基づく低関税輸入枠[「カレントアクセス」は生乳換算で13.7万トン](https://www.jacom.or.jp/niku/closeup/2025/250203-79299.php)だが、それ以上の輸入には**高い関税**がかかる。

例えるなら、バターの輸入制度は「入場者数が厳しく制限されたコンサート会場」のようなもの。チケット（輸入枠）は限られており、当日券（枠外輸入）は法外に高い。

### 2025〜2026年度の最新状況

| 指標 | 数値 |
|------|------|
| 2025年度 全国生乳生産見通し | 約729万トン（前年比99.2%） |
| 都府県の生乳生産 | 305.4万トン（300万トン割れ目前） |
| 2026年度 バター在庫見込み | 年度末3.3万トン |
| 2026年度 カレントアクセス輸入枠 | 13.7万トン（生乳換算） |
| 追加輸入の判断 | 「欠品にも過剰にもならない」として最低数量に据え置き |

[農林水産省は2026年度のバター輸入枠を最低数量に設定した](https://www.maff.go.jp/j/press/chikusan/c_gyunyu/260130.html)が、「生産動向の見通しが難しい」として需給動向の注視が必要としている。バター需要は業務用を中心に堅調であり、今後も供給の綱渡りが続く可能性が高い。

## 今後の展望 — バターの未来

### 世界市場は成長を続ける

[世界のバター市場は年率4.74%で成長を続け、2031年には619.9億ドル規模に達する見込みだ](https://www.mordorintelligence.com/industry-reports/butter-market)。特にアジア太平洋地域での需要拡大が顕著で、所得向上に伴う食生活の西洋化がバター消費を押し上げている。

### 日本の酪農は岐路に立っている

日本は「国内生産600万トン体制の維持」か「輸入依存への転換」かという岐路に立っている。現在の政策ドリフト（方針の漂流）が続けば、日本の酪農セクターは現在の半分程度にまで縮小する可能性がある。

乳牛のホルスタイン雌の出生数が減少し続けている以上、生乳生産量の回復には時間がかかる。酪農家にとって和牛生産のほうが収益性が高い現状では、「バター用の乳牛を増やそう」というインセンティブが働きにくい。

### 植物性バター代替品の台頭

世界的なヴィーガン・環境意識の高まりを背景に、植物性バター代替品の市場が拡大している。ココナッツオイル、アーモンド、カシューナッツなどを原料とした製品が登場し、従来のマーガリンとは異なる「プレミアム植物性バター」というカテゴリーが生まれつつある。

### 消費者にできること

- **適量を楽しむ**: バターは1日10〜15g程度を目安に
- **種類を使い分ける**: 普段は有塩バター、お菓子作りには食塩不使用バター、健康を意識するならグラスフェッドバター
- **国産バターを応援する**: 日本の酪農を支えるためにも、価格だけでなく品質や産地で選ぶ視点を持つ
- **植物油とバランスよく**: 日常的にはオリーブオイルなどの植物油を、特別な場面ではバターを

## まとめ — バターとの賢い付き合い方

バターは紀元前2000年から人類とともに歩んできた、最も古い加工食品の一つだ。その歴史は医薬品や化粧品から始まり、権力の象徴を経て、現代の食卓に欠かせない存在となった。

| テーマ | ポイント |
|--------|---------|
| 歴史 | 紀元前2000年から続く人類最古の加工食品の一つ |
| 製造 | 基本原理は数千年前から変わらない（クリームを攪拌して脂肪を集める） |
| 種類 | 発酵/非発酵、有塩/無塩、グラスフェッドなど目的に応じて選べる |
| 栄養 | 脂質の吸収率95%、ビタミンA・D・Eが豊富 |
| 健康 | 適量なら問題なし。植物油と併用するのがベスト |
| マーガリン比較 | どちらも一長一短。日本人のトランス脂肪酸摂取は基準以下 |
| 料理科学 | メイラード反応で色・香り・味を同時に向上させる |
| 世界市場 | 年率4.74%成長、2031年に約620億ドル規模へ |
| 日本事情 | 構造的な不足問題、高関税の輸入制度、酪農家の減少 |

バターは「敵」でも「万能薬」でもない。歴史と科学を知り、適切な量を、適切な場面で使うことが、バターとの最も賢い付き合い方だ。
