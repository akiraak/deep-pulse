## わーい!!!((歓喜)) {#acabbc77-43a5-405c-b888-f472d299fca2 name="acabbc77-43a5-405c-b888-f472d299fca2"}

##    {#1a5a9461-e20c-42d5-9a20-c46d94e00026 name="1a5a9461-e20c-42d5-9a20-c46d94e00026"}

\

Bialystocksの曲はどの曲も素晴らしいんですが、\
特に今回の「言伝」は個人的にドンピシャでして。\
\
\
初めて聴いたとき、なぜだか涙が止まらなくなりました。\

Bialystocks好きの方はもちろん、Bialystocks初めましての方にもぜひ聴いていただきたい...!\
というわけで(笑)、どんな曲なのかご紹介したいと思います!!\

\

------------------------------------------------------------------------

\

まずは冒頭。

\

<figure id="c0b6794d-0c59-49a6-a094-e9460bd34075">
<blockquote>
<p>いつか見た風景 呆然と<br />
いずれ見える流星<br />
いつも見ていたい</p>
</blockquote>
</figure>

\
先生(菊池さん)のピアノに始まり、監督(甫木元さん)のふわっとした歌声が響きます。\

ドラムだけでなく、アコギ(?)のキュッと言う音も曲のリズムを作っている感じがするのが面白いです。

\

もう一度同じメロディーを繰り返すのですが、「終演」、「運命」、「いつも見えていない」と、歌詞が少し寂しさを感じるものになっています。

\

続くこちらの歌詞。\
\

<figure id="c4ef22a2-2b8d-49d2-9156-34d05805d55b">
<blockquote>
<p>過去を掴み捨てて未来<br />
出会いし続けて 目を逸らしたい<br />
あの自分よりも 時空よりも</p>
</blockquote>
</figure>

\

なんだろう、すごく頑張って前を向いている感じ。\
過去を振り返らないことで、心を守っているのかな。

\

そして迎えるサビ...!!\

\

<figure id="0eb55989-4b7e-45c5-b93a-34955f1f94ee">
<blockquote>
<p>闇に日が差して手を伸ばしたい<br />
あなたと今 話したい<br />
朝日を見て 染まりだす<br />
もう自由に もう自由に</p>
</blockquote>
</figure>

\

\

\

\

## あーーーーーすきだーーーーー {#5a377cf9-8cf8-4f0f-9867-362df0170f85 name="5a377cf9-8cf8-4f0f-9867-362df0170f85"}

\

\

\

\
すみません、突然愛を叫んでしまいました。笑\
\
\
この「あなたと今 話したい」の、きゅっと胸が切なくなるメロディーが好きです...!!

\

サビを繰り返すのですが、\

\

<figure id="fcdbe904-5cda-4503-9e13-338484c2e622">
<blockquote>
<p>わたしは今 目を覚ました<br />
あなたと今 話したい<br />
愛すべき日が 染まり出す<br />
もう自由に 自由に 飛ぶ</p>
</blockquote>
</figure>

\

目を覚ました「わたし」。

\
ここでもう一度「あなたと今 話したい」が出てきます。

......。

\

...。

\

\

\

## あーーーーーすきだーーーーー(2回目) {#b023ee16-30ef-44ab-a670-495f09e70b66 name="b023ee16-30ef-44ab-a670-495f09e70b66"}

##   {#4bc0f992-829b-4237-821b-cc29a3cf742a name="4bc0f992-829b-4237-821b-cc29a3cf742a"}

\

\

\
何回もすみません。笑

\

\

ここに入る前の「目を覚ました」の裏で、伴奏が下降系で動くのがお洒落だし、そこに追い打ちをかける監督の切ない歌声がたまらなくて...!!

\

\
続く「もう自由に もう自由に 飛ぶ」は、ふわっと軽く、遠くへ広がっていくような歌い方で、苦しさから放たれ自由になれた「わたし」を表現しているような気がします。\
\
\

\

------------------------------------------------------------------------

\

\
そして2番へと続くのですが、実はこの『言伝』、アニメ『違国日記』のエンディングテーマです。

\
アニメから『言伝』を知ったよ!という方もたくさんおられると思うのですが...

\

\

## 『違国日記』と曲が合いすぎでは? {#3db5f022-253f-41ee-90a7-afb884f52066 name="3db5f022-253f-41ee-90a7-afb884f52066"}

\

\

\

\

曲全体に流れるふわふわとした感覚が、アサとマキオちゃんの関係性を表現しているようにも思えます。\
\
\

親子でも友達でもない二人が、適度な距離感を保ちながら、寄り添い、時には離れながら生きていく感じ。

\
\

...ごちゃごちゃと言いましたが(笑)、とりあえず、まだ『言伝』を聴いたことがないよという方は、ぜひ一度聴いてみてください!!

\

\

\
1番だけでなく、間奏部のトランペットやピアノ(シンセ?)の演奏も雰囲気があってめちゃくちゃ恰好良いですし、ラストにかけて繰り広げられる、転調×楽器隊の変化型伴奏×監督の「あなたと今 話したい」のアレンジ、という怒涛の畳み掛けも必聴です!!笑\
\
\
\

また、MVの映像も、ぜひぼんやりと眺めていただきたいです。\
(ネタバレ?になってしまいますが、少しずつ空が白んでいくんです!! ...素敵!!)

\

\

2026年の始まりに、こんなに素敵な曲に出会えて、本当に幸せだなと思います。\
いい1年になるといいなぁ。

\

\
ではまた!
