さて、今回はとうとう来た「違国日記」の最終回の感想を書いていきたいと思います。\
それでは続きからどうぞ。\
\

**第13話(最終回)「朝(あした)が来る」**\
絵コンテ,演出:大城美幸/作画監督:狩野都,大槻菜穂,豊田菜々子,他6名\
\
**[考えた先にある「考えない」行動は、間違いなく小さくても世界を変える事に繋がるんだなぁ...と「朝の歌」で感じた所に、このアニメのラストの軸を見て感じました。]{style="font-size: medium;"}**\
\
という事で、朝の軽音部の歌がそのまま、ここまでの朝を自己肯定して「思春期の答え」に綺麗になっていた話だったと思います。\
自分が表舞台に立って歌う事に理由を見出すなんて、緊張や色々な感情が立つだけで出る中で無理な話だと思うんですが、それを千世の「人生終わったって思った？」への回答として意味を持たせる...恐らく朝自身の「これまでの人生と、未来に向けての自分への肯定」も兼ねていたと思うのですが、**そこも含めて人の為に青臭く行動できる事の\"尊さ\"を見た気がしました。**\
\
確かに槙生の言う通り、自分のした事で他人の心が動く、なんて事はある事ではないのはそうですし、基本自分の為に書いているのはそうなんですよ。僕の話になりますが、この感想記事も結構需要がある事が分かっても、基本自分の備忘録みたいな感じで記事は書いてますしね。\
ですがそれを分かった上で他人の為に動く、というのならそれは「尊い」ものとなる、というのもまた真理なんですよね。**それは前提を覆そうとする、という青臭い一つの革命とも言えるわけなので。**\
\
なので、あそこでの朝の歌唱は「ソナーレ」のワンコーラス分だけでしたけど、それでも朝のこれまでの今、これからが伝わってきたようで、凄く心震えた瞬間になっていたと思います。(オアシスでみんなに水をあげるように歌っていたあのイメージ図もそうです。朝もここまで来る事ができたんだなと)\
「両親の死」という経験を高校に入る前に経験してしまって途方に暮れて、自分を見失っていた事にも気付いていない中で、**それでもなりたい自分になりたいと足掻いて前を向いた瞬間が、あの歌唱に詰まっていたと思うので。**\
目的であった千世の心を動かす事も、あれを見る限り成功していたと思うので、劇中の人物の感情と視聴者の感情が一致した瞬間を感じる事ができた意味でも良い経験だったと思っています。\
\
後、「緊張しない為には？」という朝の質問の答えに「考えて考えて、考えない」というのも良い答えだな、と思った次第です。\
この時の朝は分かっていなかったと思うのですが、散々色々な意味を考えた先に見えそうな答えに向かって、「考えずに突っ走っていく」事が、実は1番自分の感情に従っていて良い、というのはあると思いますし。\
\
そして1話冒頭の瞬間は**最終回でのライブ後の朝と槙生の世界線でしたね。**\
朝の中に自分があるからこそ槙生に対しても結構言うようになったり、朝ご飯を槙生の分も含めて作るようになって、「両親の誕生日」ともケーキを用意してあげる事で向き合う事ができた。\
それを振り返る瞬間が10年後のえみりと朝として描写されていて、その青臭さに恥ずかしがりながらも**槙生との関係性は、一緒に住まなくなったとしても続いていると。**\
\
月並みな言い方にはなりますけど、槙生と朝の関係性は、叔母と姪の関係だったとしても、**一つ「家族」の一つの形を見出していたと思います。**\
槙生が友達と接する時と違って、朝に言葉を選びまくっていたのは、朝に対しての対応を必死に考えていたから、というのはありますし、それ自体がもう「家族」に対してのそれだと思いますし、朝もそれを受け取ったと思いますしね。\
この成長の姿を15年も続けてきた実里に対して素直に尊敬していた槙生もまた、朝に向き合おうとしているからこそ大変さが伝わった、というのもありますし。これからも良い関係を続けてくれると思います。\
\
\
というわけで統括ですが、**ここまで思春期に対して向き合っていたアニメはなかった、という事に尽きますね。**\
最後の朝の歌唱もそうですけど、一人の子供が親離れをして、自分を持って、前を進んでいくという「子供から大人への移行(＝思春期)」をここまで視聴者に共感させながら克明に描き切った事の凄みを随所で感じられたので。\
\
「子供にとって親は第一の大人」「人間誰しも自分が好き」という大前提を描きつつ、でも最後には「その真理を知っても尚行動できる事は凄く尊い事だ」と締めたのは、凄くテーマ性としても綺麗だったと思いますし、このアニメはそれを真正面から向き合っていたと思います。\
なので、最初は後追いで録画でこのアニメを見てましたけど、途中からBS朝日の方でリアタイ視聴して感想を書くようになりましたからね。(投稿自体は投稿予約して朝9時にしてましたし、この記事もそうなってますけど)\
\
槙生が小説家で物書きだからこそ、創作ではないものの12年間物書きを続けてきた僕にとっては結構考えが共通するものがあったのもそうですしね。まあ槙生程僕は「自分」が強くないですし、確固たるものを持ってるわけではないですけども。\
なので、丁度30という年齢になる前、なったタイミングでこのアニメを見る事ができたのは、自分にとっても良かったと思いますし、客観的に見ても力は相当あったと思いますね。\
「私を構成する」に十分選んでも良い位には、です。\
\
**[というわけで全13話、走り抜けてくださったスタッフの方々はお疲れ様でした。そしてありがとうございました。]{style="font-size: medium;"}**
