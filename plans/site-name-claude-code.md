# サイト名に Claude Code 生成であることを明示する

## 変更対象

`src/render.ts` のみ

## 変更内容

### 1. サイト説明文の変更 (L46)

- 現在: `"ニュースを深く考察した記事を配信"`
- 変更案: `"Claude Code によるAI生成記事"`
- OGP の description やメタタグに反映される

### 2. ヘッダーにサブタイトル追加 (L198-201)

- サイト名「DEEP-PULSE」（CSS uppercase で大文字表示）の下に、小さくサブタイトルを表示
- 例: `Powered by Claude Code` や `AI-generated by Claude Code`
- CSS に `.site-header .site-subtitle` スタイルを追加

### 3. ページタイトルはそのまま

- 「DEEP-PULSE」というブランド名自体は変えず、説明文とサブタイトルで Claude Code 生成であることを伝える

## 変更しないもの

- `SITE_NAME` 定数（OGP の `og:site_name` 等に使われるため、シンプルな「deep-pulse」を維持）
- サーバー側のコード (`server.ts`)
- 記事本文のフォーマット

## 確認事項

- サブタイトルの文言は `Powered by Claude Code` でよいか？他の案があれば指定してください
- サイト説明文 `"Claude Code が生成するニュース深掘り記事"` でよいか？
